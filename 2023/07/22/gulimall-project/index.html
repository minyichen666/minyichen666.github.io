Troubleshooting

p18

  1. Error: Property ‘ Sqlsessionfactory ‘ or ‘ sqlsessiontemplate ‘ is required

Solution: Downgrade Spring boot from 3 to 2, and Downgrade Spring Cloud to corresponding version, too. This is because Spring boot 3 does not support Mybatis Plus.

p21

  1. On Windows, launching nacos/bin/start.cmd keeps showing Nacos is starting.

Solution: run start.cmd in terminal with startalone mode

  1. java.lang.IllegalArgumentException: The specified key byte array is 0 bits which is not secure enough for any JWT HMAC-SHA algorithm.

Solution: Go to nacos/config/application.properties update “nacos.core.auth.default.token.secret.key” using generated secret key from https://www.sojson.com/base64.html

p22

  1. Error Caused by: java.lang.IllegalStateException: No Feign Client for loadBalancing defined. Did you forget to include spring-cloud-starter-loadbalancer?

Solution: This is because Spring Cloud Feign start using spring-cloud-loadbalancer instead of ribbion. Therefore, add spring-cloud-loadbalancer dependency to your pom file.
